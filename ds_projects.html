Predicting Successful SpaceX Rocket Landings

My IBM Data Certificate Capstone project involved predicting whether a SpaceX launch would be able to successfully reuse a rocket so my client , SpaceY, could bid against SpaceX. I began by using the SpaceX API and webscraping to collect the data on past launches. From there I used SQL calls to an IBM DB2 server to perform analyses on the maximum payloads and most successful boosters. Using Python (Pandas), I wrangled, cleaned, and performed EDA on the data, which I summarized in a web dashboard. After "one hot encoding" the categorical values, I ran cross-validation tests on four machine learning models (logistic regression, SVM, KNN, and decision trees) to find the best performers.
